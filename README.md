### 🚀 Featured Projects  

#### 🧭 [SkillMap](https://github.com/anoushkawasthi/SkillMap)
A dynamic **learning tracker and skill visualizer** that maps users’ technical journeys through intelligent recommendations and structured progression.  
*Tech:* MERN + Next.js (TypeScript) • Node.js • Express • MongoDB  
> 🔗 _Optimized backend (skillmap_engine) for modularity, reusability, and scalability._

---

#### 🍽️ [IntelliDine – Restaurant Management SaaS](https://github.com/anoushkawasthi/intellidine)
A **production-ready, microservices-based restaurant platform** powering QR orders, analytics, and real-time operations across tenants.  
- 9 Microservices (NestJS + FastAPI)  
- Kafka-based event streaming  
- Multi-tenant PostgreSQL + Prisma ORM  
- Real-time KDS and Razorpay payments  
- 10,000+ lines of structured documentation  

> _“End-to-end flows working, 52/52 endpoints tested, and ready for scale.”_  

📊 **Stack:** NestJS • Kafka • PostgreSQL • Redis • Docker • Node 20  

---

#### 📘 [JEE Question Bank Generator](https://github.com/anoushkawasthi/jee-question-bank-generator)
An **AI-powered RAG system** that generates fresh, well-balanced JEE exam papers — not by random selection, but through **retrieval, augmentation, and validation.**  
- LLM-powered question generation (Mistral / Llama / GPT)  
- SymPy-based math validation  
- FAISS Vector Store for semantic retrieval  
- Auto-generated PDFs with detailed analytics  

📦 **Tech:** FastAPI • FAISS • MongoDB • LLMs • Regex • ReportLab
